# Project Description
An analysis of historical NCAA Basketball Tournamnets (1985-2016) based upon team and seeding.

# Research Questions to Answer
- How does each tournament seed (1-16) typically perform and are there teams that outperform their expection?
- Which games can we expect to be interesting? (Close results, high-seeds, overtime games, nearby seeds)
- How have specific teams historically performed? (Summary statistics by team)
- What games are most discussed on Twitter? Do those match what we indicated as exciting games?

# Datasets to be Used
- https://data.world/sports/ncaa-mens-march-madness
- https://data.world/mkearney/ncaa-mens-cbb-teams/

# Breakdown of Tasks
- Import dataset into Pandas datframe and clean
- Summarize performance by seed
- Summarize performance by team
- Group games according to "interesting factors"
- Identify methodology for finding Twitter Data
- Compare Twitter data to "interesting factors"
- Build interface to call on seed/team
